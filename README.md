# QWERTY Reflex Arena

A small browser game focused on keyboard speed and accuracy.

## Run locally

Because this is a static app, serve the folder with any HTTP server:

```bash
python3 -m http.server 4173
```

Then open `http://localhost:4173`.

## Gameplay

- Start with 3 lives and 30 seconds on the clock.
- Press the target key shown in the center panel.
- Correct hits increase your combo and score.
- Wrong keys or missed targets cost one life and reset the combo.
